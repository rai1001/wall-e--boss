# Seguridad y privacidad
- No grabar sin consentimiento explícito; indicar propósito en una frase.
- Minimizar scopes OAuth (calendar.readonly).
- No guardar claves reales en repositorio; usar `.env`.
- Separar datos mock de producción; permitir borrar datos de usuario.
- Aprobar antes de mover eventos/tareas de terceros.
