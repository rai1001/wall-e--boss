# Domain model
- Task: id, title, priority (VIP/Important/Normal), tags, due_date, status (pending/in_progress/done), timestamps.
- Event: id, title, start/end, location, source (mock/google), flags (is_event_day, is_off_day).
- OffDay: date, reason.
- Briefing: agenda, plan summary, dogs, house, vip list, flags (mode, off_day_suspected).
- Plan: Plan A/B/C con bloques y razones.
