# UX Planning Views
- Pantalla "Hoy": briefing, Plan A/B/C, lista de tareas, botón "Hablar".
- Vista semanal: grid/lista por día, sin drag&drop en v0.
- Formularios: creación/edición rápida (título, prioridad VIP/Important/Normal, etiquetas, due date).
- Mobile-first: tarjetas apiladas, botones táctiles >= 44px.
