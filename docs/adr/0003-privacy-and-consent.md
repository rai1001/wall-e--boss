# ADR 0003 - Privacidad y consentimiento
## Decisión
Pedir consentimiento explícito para grabar y para acciones sensibles; explicar en una frase el propósito.
## Contexto
WALL-E opera en entorno familiar/trabajo; ética primero.
## Consecuencias
- Requiere UI/UX clara para permisos.
- Evita automatizar sin aprobación de Rai.
